# Setup

## Clone Repository

```bash
git clone https://github.com/navidjp1/TweetStockSentiment.git
```

## Create Python Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### macOS/Linux
```bash
source venv/bin/activate
```

### Windows
```bash
venv\Scripts\activate.ps1
```

## Install Packages

```bash
pip install -r requirements.txt
```
